# Trader Performance vs Market Sentiment

## Objective
To analyze how Bitcoin market sentiment (Fear vs Greed) relates to trader behavior and performance.

## Datasets
- Bitcoin Fear & Greed Index
- Historical Hyperliquid trader data

## Methodology
- Cleaned and aligned datasets at a daily level
- Engineered metrics such as daily PnL and trade frequency
- Compared trader behavior across sentiment regimes
- Segmented traders into frequent and infrequent groups

## Key Insights
- Outperformance
- Consistency
- Volatility
- Asymmetry

## Strategy Recommendations
- Prioritisation
- Filtering
- Risk-limiting
- Segmentation
